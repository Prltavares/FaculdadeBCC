# Copa 2026 Tracker

Web app mobile-first em Next.js, React, Tailwind CSS e Lucide React para acompanhar seleções da Copa do Mundo de 2026.

## Estrutura principal

```txt
appworldcup26/
  app/
    globals.css
    layout.js
    page.js
  package.json
  tailwind.config.js
  postcss.config.js
  next.config.js
```

## Como rodar

```bash
npm install
npm run dev
```

Depois abra `http://localhost:3000`.

> Observação: é necessário ter Node.js e npm instalados.
